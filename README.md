# ATLAS Prime 1.0

ATLAS Prime 1.0 is de eerste MVP-demo van ATLAS.

Kernmodules:
- Dashboard Vandaag
- Financial Identity
- Doorstromer-module
- Woningwaarde Scan
- Data Engine
- Document Vault
- ATLAS Score
- Confidence Score
- Woningdetail
- Match Radar 2.0
- Scenario Simulator
- Biedklaar Paspoort
- Makelaarsdashboard
- Funda-laag
- ATLAS Copilot

Vercel:
Framework: Vite
Build command: npm run build
Output directory: dist
